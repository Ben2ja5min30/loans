Share & Access Instructions

How to access on any device
This file works on any device — phone, tablet, laptop, desktop.
No internet required after the file is downloaded. All data saves in the browser.
Option 1 — Google Drive (recommended):
1. Upload LoanSystem.html to Google Drive
2. Right-click → Share → Copy Link
3. Send link via WhatsApp, email, or SMS
4. Recipient opens in browser, saves to phone

Option 2 — WhatsApp / Email the file:
1. Send LoanSystem.html directly
2. Recipient opens the file in their browser
3. Works fully offline after opening

Option 3 — GitHub Pages (permanent URL):
1. Go to github.com → New repo → Upload file
2. Settings → Pages → Enable
3. Get permanent URL: yourname.github.io/loans

Option 4 — Netlify Drop (instant URL, free):
1. Go to app.netlify.com/drop
2. Drag & drop the HTML file
3. Get instant URL in 30 seconds — works everywhere
Important notes about data storage
Data is now synced through Supabase when the database tables exist.
Changes from one device can reflect on another device using the same live link.
For a shared system where multiple people add data:
— All users share the same live database
— New loans and payments sync across devices
— Export/Import is still available as a backup
Device compatibility
Android phones — Open in Chrome. Add to Home Screen for app-like experience.
iPhones / iPad — Open in Safari. Tap Share → Add to Home Screen.
Laptops / Desktops — Open in any browser. Works in Chrome, Edge, Firefox, Safari.
